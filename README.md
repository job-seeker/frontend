## App Components
This app is comprised of the following components:
```
<App />
  <Provider />
    <BrowserRouter />
      <Route />
        <AuthRedirect />
        <NavBar />
        <HomePage />
          <AboutUsModal />
          <AuthLanding />
            <DashboardNav />
            <CompanyListings />
            <JobListings />
            <EventListings />
              <MobileListingTable />
              <ListingTable />
                <Company>
                <CompanyView />
                  <CompanyModal />
                  <ContactModal />
                    <ContactViewModal />
                  <EventModal />
                    <EventViewModal />
                  <Job Modal />
                    <JobViewModal />
```
### User Features
* Sign up
* Log In
* Create a company
* Create or delete jobs associated with a company
* Create or delete events associated with a company
* Create or delete contacts associated with a company
* Log out
